# Treasure Island Game

(PT/BR) Criação de um mini jogo utilizando a linguagem de Python. Esse exercício vem do curso "100 Days of Code™: The Complete Python Pro Bootcamp", da professora Dr. Angela Yu, disponível na plataforma Udemy. O jogo tem sua versão em Inglês ou Português. 

(English) Creation of a mini game using the Python programming language. This exercise comes from the course "100 Days of Code™: The Complete Python Pro Bootcamp" by Dr. Angela Yu, available on the Udemy platform. The game has versions in both English and Portuguese.
